# 컴퓨터 시스템 강의계획서

## 강의 정보
- 과목명: 컴퓨터 시스템
- 대상: 산업공학과 학부생
- 학점: 3학점
- 강의시간: 주 3시간

## 강의 목표
산업공학도에게 필요한 컴퓨터 시스템의 기본 개념과 실무 능력을 배양합니다. Python 프로그래밍을 통해 컴퓨터 구조, 운영체제, 데이터베이스, 네트워크 등의 핵심 개념을 학습합니다.
